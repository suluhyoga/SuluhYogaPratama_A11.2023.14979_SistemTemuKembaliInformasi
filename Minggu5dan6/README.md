Minggu 5&6 - Word2Vec  

Pada latihan ini, saya menerapkan konsep Word Embedding menggunakan Word2Vec untuk merepresentasikan kata ke dalam bentuk vektor numerik berdasarkan konteks kemunculannya dalam teks. Proses dimulai dengan melatih model Word2Vec pada dataset teks, kemudian menganalisis hubungan semantik antar kata melalui perhitungan kemiripan serta visualisasi embedding menggunakan PCA dan t-SNE. Tujuan dari latihan ini adalah untuk memahami bagaimana Word2Vec mampu menangkap makna dan kedekatan antar kata dalam ruang vektor berdimensi rendah.
