Minggu 9 - Klasifikasi Text Mining Naive Bayes dan KNN  

Pada latihan ini, saya menerapkan metode klasifikasi teks untuk analisis sentimen menggunakan algoritma Naive Bayes dan K-Nearest Neighbors (KNN). Proses dimulai dengan preprocessing teks, meliputi pembersihan data, tokenisasi, stopword removal, dan stemming, kemudian dilanjutkan dengan ekstraksi fitur menggunakan TF-IDF. Tujuan latihan ini adalah untuk membandingkan performa dua algoritma klasifikasi dalam text mining berdasarkan metrik accuracy, precision, recall, dan F1-score.
