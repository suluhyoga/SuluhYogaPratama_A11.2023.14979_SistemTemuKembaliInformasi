Minggu 8 - Naive Bayes  

Pada latihan ini, saya menerapkan konsep klasifikasi teks untuk menganalisis sentimen menggunakan algoritma Multinomial Naive Bayes. Proses diawali dengan tahap preprocessing teks seperti pembersihan data, tokenisasi, stopword removal, dan stemming, kemudian dilanjutkan dengan ekstraksi fitur menggunakan TF-IDF. Tujuan dari latihan ini adalah untuk memahami alur kerja dasar text mining, mulai dari pengolahan teks hingga evaluasi performa model klasifikasi berdasarkan metrik accuracy, precision, recall, dan F1-score.
