Minggu 10 - Document Clustering  

Pada latihan ini, saya mempelajari konsep clustering dokumen untuk mengelompokkan teks berdasarkan kemiripan kontennya tanpa menggunakan label. Dokumen direpresentasikan dalam bentuk vektor menggunakan TF-IDF, kemudian dikelompokkan menggunakan algoritma K-Means. Tujuan latihan ini adalah untuk memahami bagaimana dokumen dengan topik serupa dapat dikelompokkan secara otomatis serta bagaimana hasil clustering dapat dianalisis dan divisualisasikan.
