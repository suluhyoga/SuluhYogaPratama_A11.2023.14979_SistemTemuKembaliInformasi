Minggu 13 - Topic Modelling  

Pada latihan ini, saya menerapkan Topic Modelling menggunakan algoritma Latent Dirichlet Allocation (LDA) untuk menemukan topik-topik utama dalam kumpulan dokumen teks. Proses dimulai dari preprocessing teks, pembuatan dictionary dan corpus, hingga pemodelan topik dan penentuan dominant topic pada setiap dokumen. Tujuan latihan ini adalah untuk memahami bagaimana LDA dapat mengidentifikasi tema utama yang tersembunyi dalam data teks secara otomatis.
